# dotfiles

## Installation

installコマンドを叩く前に oh-my-zsh を入れておこう
https://shinshin86.hateblo.jp/entry/2022/10/30/000000

### for Mac/Linux

```sh
$ cd ~/
$ git clone https://github.com/nakahashi/dotfiles.git
$ dotfiles/install.sh
```

### for Windows

```bat
$ cd %HOMEPATH%
$ git clone https://github.com/nakahashi/dotfiles.git
$ dotfiles\install.cmd
```
